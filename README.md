# Sales_Predciton_Model
 Sales Prediction Project Summary:Data Prep: Cleaned a sales CSV in Colab using Pandas; split features (Price, Marketing_Spend) from target (Sales) into train/test sets.Modeling: Tested 3 architectures using $R^2$ &amp; RMSE.Results: Linear Regression won ($R^2$: 0.83, RMSE: 295), beating Random Forest (0.82) and a weak standalone Decision Tree (0.63).
